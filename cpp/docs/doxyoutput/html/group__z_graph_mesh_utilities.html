vector< float > &  dist,
vector< int > &  parent

This method computes the shortest path from the source vertex to all vertices of the zGraph/zMesh.

based on Dijkstra’s shortest path algorithm (https://www.geeksforgeeks.org/dijkstras-shortest-path-algorithm-greedy-algo-7/)

Template Parameters
T- Type to work with zGraph and zMesh.
Parameters
[in]inHEDataStructure- input graph or mesh.
[in]index- source vertex index.
[out]dist- container of distance to each vertex from source.
[out]parent- container of parent vertex index of each to each vertex. Required to get the path information.
Since
version 0.0.1

This method computes the shortest path from the source vertex to destination vertex of the zGraph/zMesh. The distance and parent containers need to be computed before using the shortest distance method.

Template Parameters
T- Type to work with zGraph and zMesh.
Parameters
[in]inHEDataStructure- input graph or mesh.
[in]indexA- source vertex index.
[in]indexB- destination vertex index.
[in]dist- container of shortest distances to each vertex from the source. To be computed using the shortest distance method.
[in]parent- container of parent to each vertex. To be computed using the shortest distance method.
[in]type- zWalkType - zEdgePath or zEdgeVisited.
[out]edgeContainer- container of edges of the shortest path(zEdgePath) or number of times an edge is visited(zEdgeVisited).
Since
version 0.0.1
