int zSpace::minDistance ( vector< float > &  dist,
vector< bool > &  sptSet 
)
inline

This method returns the vertex with minimum distance value, from the set of vertices not yet included in shortest path tree. To be used with shortestDistance method for mesh/graph.

based on Dijkstra’s shortest path algorithm (https://www.geeksforgeeks.org/dijkstras-shortest-path-algorithm-greedy-algo-7/)

Parameters
[out]dist- container of distance to each vertex from source.
[out]sptSet- container of shortest path tree for each vertex..
