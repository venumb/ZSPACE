This method computes the shortest path from the source vertex to all vertices of the graph.

based on Dijkstra’s shortest path algorithm (https://www.geeksforgeeks.org/dijkstras-shortest-path-algorithm-greedy-algo-7/)

Parameters
[in]inGraph- input graph.
[in]index- source vertex index.
[out]dist- container of distance to each vertex from source.
[out]parent- container of parent vertex index of each to each vertex. Required to get the path information.
